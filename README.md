# Personal Projects

This repository is the parent directory for multiple full-stack web application projects that I am building.

Each project lives in its own subdirectory and includes its own `README.md` with project-specific details such as:

- project overview
- features
- tech stack
- setup and installation
- environment variables
- run instructions
- API details
- deployment notes

## Projects

| Project        | Description                                                                          | Status  |
| -------------- | ------------------------------------------------------------------------------------ | ------- |
| `Local_Chatbot` | Full-stack LLM + RAG chatbot boilerplate with Next.js, FastAPI, LangGraph, and Qdrant. | Scaffolded |
| `Trust_System` | Full-stack web application for building and experimenting with a trust-based system. | Planned |

More projects will be added over time.

## How To Use This Repository

1. Open the project folder you are interested in.
2. Read that project's `README.md`.
3. Follow the setup and run instructions provided inside the subproject.

## Notes

- This root `README.md` is only meant to provide a high-level overview of the repository.
- Each subproject is intended to be self-contained.
- Project-specific documentation will be maintained inside each subdirectory.
